# Bike Buyers Data Analysis (Interactive Dashboard Creation Using MS Excel)
## Project Objective
The objective of this project is to analyze customer purchasing behavior using a bike buyers dataset and create an interactive dashboard in Microsoft Excel. This analysis helps understand the factors influencing bike purchases and supports better business decision-making through data-driven insights.
## Dataset used
- <a href="https://github.com/anand5543/Data-Analysis-Dashboard/blob/main/Bike%20Buyers%20Data%20Analysis%20project.xlsx">Dataset</a>

## Questions (KPIs)
•	Which age group is more likely to purchase bikes? 
•	Compare bike purchases based on gender. 
•	What is the average income of customers who purchased bikes? 
•	Which occupation group purchases more bikes? 
•	How does marital status affect bike purchases? 
•	Relation between commute distance and bike purchases. 
•	Which education level contributes more to bike purchases? 
•	Analyze regional trends in bike purchases.

## Process
•	Verified dataset for missing values and inconsistencies. 
•	Cleaned and formatted the data to maintain consistency. 
•	Performed data preprocessing and categorization. 
•	Created pivot tables based on business questions. 
•	Built charts and visualizations for better analysis. 
•	Combined all visualizations into a single interactive dashboard. 
•	Added slicers and filters to make the dashboard dynamic and user-friendly.
