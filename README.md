# Bike Buyers Data Analysis (Interactive Dashboard Creation Using MS Excel)
## Project Objective
The objective of this project is to analyze customer purchasing behavior using a bike buyers dataset and create an interactive dashboard in Microsoft Excel. This analysis helps understand the factors influencing bike purchases and supports better business decision-making through data-driven insights.
